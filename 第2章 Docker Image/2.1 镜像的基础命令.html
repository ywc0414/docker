2.1 镜像的基础命令

查询镜像
docke search redis

拉取镜像
# 当前没指定tag时，默认拉取最新的（latest）
docker pull redis

# 拉取指定版本的镜像
docker pull redis:4.0.8
查看已拉取的镜像
docker images
给镜像打tag
# docker tag 源tag 目标tag
docker tag redis:4.0.8 myredis:v1
查看镜像详情
docker inspect redis:4.0.8
查看镜像构建历史
docker history redis:4.0.8
删除镜像
# 删除镜像时，需要先删除引用了该镜像的容器
docker rmi redis:4.0.8
